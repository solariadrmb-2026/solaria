⚙️ Sobre o Sistema

☀️ O que é o Solaria?

O Solaria é um sistema web inteligente desenvolvido para o monitoramento e automação da irrigação agrícola e hortas comunitárias, utilizando tecnologias de IoT,sensores inteligentes e integração em tempo real para otimizar o uso da água e melhorar a eficiência no cultivo.

📡 Tecnologias Utilizadas

ESP32

Microcontrolador responsável pela comunicação do sistema.

Firebase

Banco de dados em tempo real para armazenamento.

HTML

Estrutura das páginas do sistema web.

CSS

Estilização e responsividade das páginas.

JavaScript

Interatividade e comunicação dinâmica do sistema.

📈 Funcionalidades

Monitoramento

Acompanhamento dos sensores em tempo real.

Irrigação

Controle automático da irrigação da horta.

Dashboard

Visualização prática das informações do sistema.

Alertas

Notificação sobre o estado dos sensores.

Responsabilidade

Compatibilidade com computadores e celulares.
